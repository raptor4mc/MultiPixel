# Server Penalty Tiers
Version 1.0 — Relimintor (Raptor4 / MultiPixel)

This document expands Sections 9, 12, 14, and 15 of the MultiPixel
Source-Available License 2.2 — Fortress Edition.

It defines the official penalty tiers for Server violations within the
MultiPixel ecosystem.

Penalties are determined by:
• violation severity
• frequency
• intent
• user harm
• financial harm
• audit findings

---------------------------------------------------------------------------

SECTION 1 — OVERVIEW OF PENALTY TIERS
---------------------------------------------------------------------------

Penalties are divided into four escalating tiers:

• **Tier 0 — Advisory Notice**  
• **Tier 1 — Corrective Action Required**  
• **Tier 2 — Temporary Suspension**  
• **Tier 3 — Permanent Delisting & License Termination**  

Severe violations may skip directly to Tier 2 or Tier 3.

---------------------------------------------------------------------------

SECTION 2 — TIER 0: ADVISORY NOTICE
---------------------------------------------------------------------------

Tier 0 applies to:
• minor violations  
• first-time administrative issues  
• accidental configuration errors  

Penalties include:
• advisory notice  
• request for correction  
• 7-day correction window  

Tier 0 does NOT affect:
• server approval status  
• monetization status  
• listing status  

Failure to correct escalates to Tier 1.

---------------------------------------------------------------------------

SECTION 3 — TIER 1: CORRECTIVE ACTION REQUIRED
---------------------------------------------------------------------------

Tier 1 applies to:
• repeated minor violations  
• moderate violations  
• incomplete logs  
• insufficient safety systems  
• untrained staff  
• unapproved cosmetic monetization  
• insecure data storage  

Penalties include:
• formal warning  
• mandatory correction  
• 72-hour correction window  
• mandatory follow-up audit  

Tier 1 may temporarily restrict:
• new monetization features  
• new plugins/mods  
• new staff appointments  

Failure to correct escalates to Tier 2.

---------------------------------------------------------------------------

SECTION 4 — TIER 2: TEMPORARY SUSPENSION
---------------------------------------------------------------------------

Tier 2 applies to:
• repeated moderate violations  
• failure to correct Tier 1 issues  
• unapproved monetization  
• incomplete safety systems  
• incomplete moderation systems  
• unlogged moderation actions  
• insecure or missing logs  
• unapproved plugins (non-malicious)  

Penalties include:
• temporary suspension of server listing  
• temporary suspension of monetization  
• mandatory full audit  
• 30-day probation period  
• increased reporting requirements  

Suspension duration:
• 7–30 days depending on severity  

Failure to comply during suspension escalates to Tier 3.

---------------------------------------------------------------------------

SECTION 5 — TIER 3: PERMANENT DELISTING & LICENSE TERMINATION
---------------------------------------------------------------------------

Tier 3 applies to severe violations, including:

### 5.1 Safety & User Protection
• allowing hate speech  
• allowing extremist content  
• allowing harassment  
• allowing sexual content involving minors  
• ignoring safety reports  
• enabling predatory behavior  

### 5.2 Moderation Abuse
• selling staff roles  
• staff abusing power  
• deleting moderation logs  
• retaliation against reporters  

### 5.3 Data & Privacy Violations
• collecting personal data  
• collecting sensitive data  
• selling or sharing user data  
• tracking users across servers  
• fingerprinting users  
• deleting logs during audits  

### 5.4 Monetization Violations
• pay-to-win systems  
• selling progression, stats, or power  
• gambling systems  
• crypto/NFT monetization  
• revenue concealment  
• deleted payment logs  
• monetizing without required approval  

### 5.5 Operational & Security Violations
• running malicious plugins  
• modifying MultiPixel client code  
• bypassing security systems  
• impersonating official MultiPixel servers  

Penalties include:
• permanent delisting  
• permanent monetization ban  
• termination of approval status  
• legal action  
• financial penalties  
• termination of license  

Tier 3 penalties are final.

---------------------------------------------------------------------------

SECTION 6 — ESCALATION RULES
---------------------------------------------------------------------------

Violations escalate based on:

• frequency  
• severity  
• intent  
• user harm  
• financial harm  
• audit findings  

Escalation examples:

• 3 minor violations → Tier 1  
• 2 moderate violations → Tier 2  
• any severe violation → Tier 3  
• repeated Tier 2 violations → Tier 3  

---------------------------------------------------------------------------

SECTION 7 — AUTOMATIC TIER 3 VIOLATIONS
---------------------------------------------------------------------------

The following ALWAYS result in Tier 3 penalties:

• sexual content involving minors  
• extremist content  
• hate speech  
• staff abusing power  
• selling staff roles  
• pay-to-win monetization  
• gambling systems  
• crypto/NFT monetization  
• revenue concealment  
• deleted logs  
• refusal to cooperate with audits  
• impersonating official MultiPixel services  
• collecting personal or sensitive data  

These violations bypass all lower tiers.

---------------------------------------------------------------------------

SECTION 8 — PENALTY REDUCTION
---------------------------------------------------------------------------

Penalties may be reduced if:

• the violation was accidental  
• the server self-reported  
• the server cooperated fully  
• the issue was corrected immediately  
• no users were harmed  
• no revenue was concealed  

Penalty reduction is at MultiPixel’s discretion.

---------------------------------------------------------------------------

SECTION 9 — PENALTY ENFORCEMENT
---------------------------------------------------------------------------

Penalties may be enforced through:

• warnings  
• suspension  
• delisting  
• monetization bans  
• legal action  
• license termination  

Enforcement actions are final.

---------------------------------------------------------------------------

All Servers must comply with the latest version.
