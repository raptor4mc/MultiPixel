# MultiPixel Governance Forms
This directory contains all official forms used within the MultiPixel
ecosystem. Forms are used to request approvals, submit mods, report issues,
and manage compliance across servers, mods, and community interactions.

All forms are organized by category:

---

## 📂 Server Forms (`Server/`)
Forms related to server operation, approval, compliance, and incident
reporting.

Examples:
- Server Approval Form
- Server Incident Report Form

---

## 📂 Mod Forms (`Mod/`)
Forms related to mod submission, approval, and monetization.

Examples:
- Mod Monetization Request Form
- Mod Submission Form

---

## 📂 User Forms (`User/`)
Forms used by players or community members.

Examples:
- Player Report Form
- Appeal Request Form

---

## 📂 Other Forms (`Other/`)
Forms that do not fit into the above categories.

Examples:
- Partnership Request Form

---

## 🧱 Naming Convention
All forms follow this naming pattern:

`[Category]_[Purpose]_Form_vX.X.pdf`

Examples:
- `Server_Approval_Form_v1.0.pdf`
- `Mod_Monetization_Request_Form_v1.0.pdf`

---

## 🔧 Updating Forms
When updating a form:
1. Increment the version number inside the file.
2. Update the filename to match the new version.
3. Add a note to the Governance CHANGELOG (if present).

---

## 📬 Submission Process
Each form includes instructions for:
- where to submit it
- what information is required
- how long approval typically takes
- what happens after submission
